## Installation

This port runs the Linux x86_64 build of the game via Box64. Wine is not involved.

Copy your Linux game files into `ports/portfolder/data/`.

If you own the Steam version, you can usually grab the Linux depot via Steam's console with `download_depot <appid> <depotid>`.
