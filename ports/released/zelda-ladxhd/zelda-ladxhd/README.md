## Installation
Place the original `Links Awakening DX HD v1.0.0.zip` file in the `zelda-ladxhd/data` folder. The port will automatically extract and patch the game to the latest version on first launch.

## Notes
- Patching requires an internet connection (patches are downloaded from the [LADXHD-Updated](https://github.com/BigheadSMZ/Zelda-LA-DX-HD-Updated) repository).
- Nintendo layout users should create a `swapabxy.txt` file in `zelda-ladxhd` and in game settings swap Confirm and Cancel for ui.
- Users with small display resolutions or odd aspect ratios can fiddle with the files in `zelda-ladxhd/data/Mods/LAHDMods` with a text editor to set ui scale to float overrides.
