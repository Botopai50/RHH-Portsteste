## Ratcheteer DX
From the creator of MinUI, Ratcheteer DX is a nostalgic love letter to Gameboy-era action/adventure puzzlers! Zelda fans will be comfortable with this!

Download the demo with `download_depot 3908650 3908653 2932770551190015228` and add to the data folder, and if you enjoy the demo purchase the full game and add its data instead!
