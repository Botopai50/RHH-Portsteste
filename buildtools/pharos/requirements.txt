# Pharos build + runtime Python dependencies.
#
# Runtime:
#   PySDL2       - Python ctypes bindings for SDL2 (used in ui.py, etc.)
#   PySDL2-dll   - On arm64 this is source-only and ignored at runtime;
#                  Pharos relies on the CFW's /usr/lib/libSDL2 (path set by
#                  the launchscript via PYSDL2_DLL_PATH).
#   certifi      - CA bundle. Minimal CFWs (Knulli) ship Python without
#                  ssl certs in the standard paths, so we bundle our own
#                  and point urllib at it from main.py.
#
# Build:
#   pyinstaller  - --onefile bundler.

PySDL2
PySDL2-dll
certifi
pyinstaller
